# 隐私说明

本说明描述招聘助手浏览器扩展会处理哪些数据，以及这些数据会存放在哪里。

## 本地处理的数据

扩展运行时可能读取当前 BOSS 直聘页面中的招聘相关内容，包括：

- 候选人列表和筛选状态。
- 简历文本、候选人信息摘要和评级结果。
- 沟通状态、打招呼/跟进/感谢等话术。
- 运行记录、处理日志和错误信息。
- 用户在设置页填写的自定义 AI 配置。

这些数据主要用于在本机完成自动化辅助、状态展示、日志记录和用户配置恢复。

## 本地存储

扩展使用 `chrome.storage.local` 保存设置和运行状态。可能保存的内容包括：

- 招聘设置、筛选项、话术模板和评级 Prompt。
- 自定义 AI Base URL、API Key、Model 和高级参数。
- 今日使用统计、处理日志和诊断日志快照。

API Key 仅保存在本机浏览器的扩展存储中。请不要把浏览器用户数据目录、扩展存储文件或导出的设置文件公开上传。

## 网络请求

扩展会访问 BOSS 直聘页面本身需要的网页和接口。

当用户配置并使用自定义 AI 时，扩展会向用户填写的 AI API 发送请求。请求内容可能包含简历文本、岗位沟通上下文、话术生成需求和评级 Prompt。扩展不内置 AI 服务，也不把数据发送到项目维护者的服务器。

## 诊断日志

用户开启诊断功能后，扩展可能记录 BOSS 直聘页面请求的 URL、方法、状态码、耗时、响应头和经过脱敏的请求片段。诊断日志用于排查登录、跳转和接口异常。

提交 issue 前，请先删除以下内容：

- 姓名、手机号、邮箱、微信号等个人信息。
- 简历原文、聊天内容和候选人唯一标识。
- Cookie、Token、Authorization、API Key 和其他密钥。
- 公司内部地址、私有接口和不可公开的业务信息。

## 用户责任

使用者需要自行确认：

- 有权处理相关候选人和招聘数据。
- 自定义 AI 服务满足自己的隐私、安全和合规要求。
- 使用方式符合目标网站服务条款和当地法律法规。
